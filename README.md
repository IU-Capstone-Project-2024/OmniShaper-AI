# OmniShaper-AI

### Enhance your 3D modeling using Deep Learning

### Requirements:
- Python 3.9
- CUDA 11.8
### Installation
```bash
git clone '...'
cd OmniShaper-AI
python -m venv venv
source venv/bin/activate
./scripts/install-requirements.sh
```

### Usage
Run `python -c "from ml_pipeline.pipeline import promt_to_3D; promt_to_3D('Your prompt')" `
